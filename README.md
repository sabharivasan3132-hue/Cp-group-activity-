Hotel Room Booking System

This is an mini project developed using C programming. The project is menu-driven and uses file handling to manage hotel room bookings.

Objective

To design and implement a Hotel Room Booking System that can manage rooms, bookings, check-in, check-out, and cancellation efficiently using C.

Features

Add new rooms
Display available rooms
Book a room
Check-in a guest
Check-out a guest
Cancel booking
File-based data storage
Technologies Used

C Programming
File Handling
